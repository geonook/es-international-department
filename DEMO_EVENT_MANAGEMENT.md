## 🎯 使用方法 | Usage Instructions

### 啟動系統 | Start System
```bash
npm run dev
# 訪問 http://localhost:3000/admin
# 使用管理員帳號登入
```

### 活動管理步驟 | Event Management Steps

1. **登入管理員後台**
   - 訪問 `/admin`
   - 使用 Google OAuth 登入
   - 確認管理員權限

2. **進入活動管理**
   - 點擊側邊欄「活動管理」
   - 或直接訪問 `/admin/events`

3. **建立新活動**
   - 點擊「新增活動」按鈕
   - 填寫活動資訊表單
   - 設定日期時間和地點
   - 選擇目標年級
   - 配置報名設定

4. **管理現有活動**
   - 查看活動列表
   - 使用篩選器找到特定活動
   - 編輯活動資訊
   - 更新活動狀態
   - 刪除不需要的活動
